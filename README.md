# 🎁 InstaDraw - 인스타 댓글 추첨기 (무료 웹앱)

누구나 간편하게 사용할 수 있는 **인스타그램 전용 댓글 추첨기(Instagram Comment Picker)** 입니다.
터미널 명령어 없이 클릭 한 번으로 실행할 수 있으며, 나만의 맞춤형 브랜드 디자인으로 이벤트를 진행해보세요! 🎉

## 💡 왜 InstaDraw인가요?
- **비개발자 친화적:** 파이썬만 깔려 있다면 복잡한 세팅 없이 더블 클릭 한 번으로 켤 수 있어요.
- **예쁜 UI/UX:** 기존의 딱딱한 추첨기들과 다르게, 마치 인스타그램 피드를 보는 듯한 깔끔하고 예쁜 디자인을 제공해요.
- **안전한 데이터:** 내 PC에서 직접 실행하기 때문에 인스타그램 토큰이나 참가자 데이터가 외부 서버로 절대 유출되지 않습니다! 🔒
- **다중 계정 및 중복 필터링:** 여러 브랜드 계정을 관리할 수 있고, "1인 1응모(중복 제외)" 기능도 기본 탑재되어 있습니다.

## 🚀 1. 준비 사항 (Python 설치)

이 프로그램을 실행하려면 컴퓨터에 파이썬(Python)이 꼭 깔려 있어야 합니다.

### 🪟 Windows 사용자
1. [Python 공식 홈페이지](https://www.python.org/downloads/windows/)에 접속합니다.
2. **Download Windows installer (64-bit)**를 클릭해서 설치 파일을 받습니다.
3. ⭐️ **가장 중요:** 설치 화면 맨 아래에 있는 **[Add Python to PATH]** 체크박스를 무조건! 선택한 뒤에 [Install Now]를 눌러주세요.

### 🍎 Mac 사용자
1. 터미널(Terminal) 앱을 엽니다.
2. `python3 --version` 이라고 치고 엔터를 눌러봅니다. 버전이 나오면 설치된 거예요!
3. 만약 설치되어 있지 않다면, [Python 공식 홈페이지](https://www.python.org/downloads/macos/)에서 다운로드해 주세요.

## ⚙️ 2. 설정 (환경 변수)

디자인을 내 마음에 쏙 들게 바꾸거나, 앱을 제대로 작동시키려면 `.env` 파일 설정이 필요합니다.
프로젝트 폴더 안에 있는 `.env.example` 파일의 이름을 **`.env`** 로 변경한 후, 메모장으로 열어주세요.

```env
# 1. 인스타그램 API 토큰 (필수 ⭐)
# Meta for Developers (페이스북 개발자 센터)에서 발급받은 Graph API Access Token을 넣어주세요.
INSTAGRAM_ACCESS_TOKEN=여기에_발급받은_토큰_붙여넣기

# 2. 브랜드 디자인 설정 (선택 🎨)
# 아래 값을 수정해서 앱 상단 이름이나 메인 컬러를 바꿀 수 있습니다.
APP_BRANDING_NAME=나만의_브랜드
# APP_EMOJI=🎁

# [UI 컬러 테마 프리셋] - 원하는 테마의 주석(#)을 지워보세요!
# 🖤 모던 블랙 (미니멀, 트렌디)
# APP_PRIMARY_COLOR=#2B2D42
# APP_BG_COLOR=#F4F6F8

# 🌿 프레시 민트 (청량함, 산뜻함)
# APP_PRIMARY_COLOR=#2A9D8F
# APP_BG_COLOR=#F0F7F4
```

> ⚠️ **테마에 대한 변명(?)**: 기본값은 핑크색 파스텔 톤과 꽃(🌸) 이모지로 되어 있습니다. 제 취향은 절대 아니고요... 와이프의 강력한 요구사항이 반영된 결과물입니다! 😂 너무 화려하다면 꼭 위 설정에서 모던 블랙이나 민트로 변경해 주세요.

## 🏃‍♂️ 3. 실행 방법 (원클릭)

### 🪟 Windows 사용자
- 폴더 안에 있는 **`run_instadraw.bat`** 파일을 마우스로 더블 클릭하세요. (끝!)
- 처음에 까만 창이 뜨면서 이것저것 설치한 뒤, 알아서 인터넷 브라우저가 열리며 추첨기가 실행됩니다.

### 🍎 Mac / Linux 사용자
1. 터미널을 열고 파일을 다운받은 폴더로 이동합니다. (`cd 경로`)
2. 처음 한 번만 실행 권한을 줍니다: `chmod +x run_instadraw.sh`
3. 프로그램 실행: `./run_instadraw.sh`
